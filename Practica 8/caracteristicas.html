Caracteristicas

Autor:Andrea Gomez Gomez

Definicion:

1. Confidencialidad Garantiza que solo las personas, sistemas o procesos autorizados puedan acceder a la información sensible. Ejemplo: El uso de contraseñas, cifrado de datos y autenticación multifactor son ejemplos de medidas para mantener la confidencialidad de la información.

2. Integridad Asegura que los datos no sean modificados, manipulados o destruidos sin autorización, y que permanezcan exactos y completos. Ejemplo: La implementación de controles de acceso y sistemas de verificación para detectar cambios no autorizados en los archivos o bases de datos.

3. Disponibilidad Asegura que la información, sistemas y servicios estén accesibles y operativos cuando se necesiten. Ejemplo: El uso de redundancias, sistemas de respaldo y planes de recuperación ante desastres que garantizan el funcionamiento continuo de los sistemas, incluso ante fallas o ataques.

4. Autenticación Proceso mediante el cual se verifica la identidad de un usuario o sistema antes de otorgarle acceso a los recursos. Ejemplo: Contraseñas seguras, autenticación biométrica (huellas digitales, reconocimiento facial) y autenticación multifactor (MFA), que requiere dos o más formas de verificación.

5. Autorización Después de la autenticación, el sistema determina qué recursos y acciones están permitidos para el usuario autenticado. Ejemplo: Un administrador tiene acceso a funciones avanzadas del sistema, mientras que un usuario común puede estar restringido a operaciones básicas.

6. Trazabilidad o Auditoría Capacidad de rastrear y registrar todas las acciones y accesos a la información dentro de un sistema. Ejemplo: Los registros de auditoría que documentan quién accedió a qué datos, cuándo y desde dónde. Esto permite identificar el origen de incidentes de seguridad.

7. Resiliencia Capacidad de un sistema o red para resistir y recuperarse rápidamente ante ataques, fallos o interrupciones. Ejemplo: El uso de sistemas de respaldo automatizados y capacidades de respuesta rápida ante incidentes para garantizar que un sistema pueda restablecerse en poco tiempo.
